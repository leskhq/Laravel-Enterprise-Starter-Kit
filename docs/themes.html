To change the default theme, set the DEFAULT_THEME variable in the .env file:

THEME.DEFAULT=red

LESK comes with 3 themes: default, green and red. Both the red and green themes inherit much of their look from the default theme which is mostly blue and based on the look of the almasaeed2010/AdminLTE Web template. For more details on how to configure and develop your own themes refer to the documentation of the yaapis/Theme package.
