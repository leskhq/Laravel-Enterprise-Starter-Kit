Advanced datatables and reports can be easily created using jqGrid and mgallegos/laravel-jqgrid.

The module Report Users provides 2 sample reports:

  • A simple user list.
  • A list of permissions and roles for each user, grouped by user and roles. This is implemented by first creating a view in the database as it would have been too complicated for Laravel Query Builder to handle.

NOTE 1: The view is created in a migration of the module.

NOTE 2: All jqGRids reports are using the base jQuery UI.
